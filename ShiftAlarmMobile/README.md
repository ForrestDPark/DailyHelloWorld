# ShiftAlarmMobile

`shift_alarm.py`(맥 메뉴바 교대근무 알람 앱)의 핵심 기능을 iOS로 옮긴 SwiftUI 앱입니다.

## 기능

- 근무표(JSON) 기반 자동 근무 인식 + 근무별 알람 시간(Day/Swing/GY) 로컬 알림
- 주간 리마인더: 헬스장(주 2회), 엄마한테 전화(휴무 시작일), 카톡 정리(휴무 마지막날),
  아울렛 쇼핑(한 달에 한 번, 그 달의 첫 번째 휴무 시작일) — 각각 설정에서 켜고 끌 수 있음
- 연차 등으로 오늘만 근무표와 다르게 지정하고 싶을 때를 위한 수동 모드

날씨/실시간 급여 표시 등 맥 앱의 나머지 기능은 이번 버전에는 포함하지 않았습니다.

## 사용 전 꼭 해야 할 일: 근무표 데이터 넣기

이 레포에는 실제 근무표 데이터(`d_team_schedule_2026.json`)가 포함되어 있지 않습니다
(맥에서 `~/.shift_alarm_config.json` 옆, `shift_alarm.py`와 같은 폴더에 있는 개인 근무표 파일이라
공개 레포에는 커밋하지 않는 게 맞습니다). 지금 들어있는 건 빈 `{}` 플레이스홀더입니다.

빌드 전에 `ShiftAlarmMobile/ShiftAlarmMobile/d_team_schedule_2026.json`을
실제 근무표 내용으로 교체하세요. 포맷은 맥 앱과 동일합니다.

```json
{
  "2026-07-17": "D",
  "2026-07-18": "S",
  "2026-07-19": "G",
  "2026-07-20": "휴",
  "2026-07-21": "휴"
}
```

코드(`D`=Day, `S`=Swing, `G`=GY, `휴`=휴무) 의미도 맥 앱과 동일합니다.

## 빌드 방법

1. 맥에서 `ShiftAlarmMobile.xcodeproj`를 Xcode로 엽니다.
2. 위 안내대로 `d_team_schedule_2026.json`을 실제 근무표로 교체합니다.
3. Signing & Capabilities에서 본인 Apple ID / 팀으로 서명을 맞춥니다
   (프로젝트에는 `hello_swift`와 같은 팀 ID가 기본으로 들어있습니다).
4. 기기 또는 시뮬레이터에서 실행하고, 알림 권한 요청에 허용을 눌러주세요.

앱을 열 때마다(포그라운드로 돌아올 때) 앞으로 60일치 알림을 다시 예약합니다
(iOS는 앱당 최대 64개의 예약 알림만 허용하므로, 근무 알람 + 리마인더를 합쳐
이 범위 안에서 관리합니다). 그래서 최소 두 달에 한 번 정도는 앱을 열어줘야
알림이 끊기지 않습니다.
